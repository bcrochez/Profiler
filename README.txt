bin contient des fichiers de log ainsi que les exécutables après compilation.

myprofiler correspond au profiler.

Arguments attendus
$> ./bin/myprofiler nom_du_fichier_de_log

test et fact servent de test et créent un fichier de log à l'aide de la macro.

doc contient la documentation générée en utilisant doxygen (html + latex).
include contient les différents headers ainsi que la macro de profilage.
src contient les sources du projet et des deux fichiers de test de la macro.
